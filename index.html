🚀 Hosting on GitHub Pages (FREE)
  1. Extract the downloaded ZIP
  2. Open js/firebase-config.js and paste your Firebase credentials
  3. Go to github.com → New repository (e.g. bachelor-house)
  4. Upload all files to the repo
  5. Settings → Pages → Deploy from main branch → Save
  6. Share yourusername.github.io/bachelor-house with everyone
  7. Each person opens the link → clicks Allow for notifications
✅ GitHub Pages is completely free. Firebase free tier supports up to 1GB storage & 50,000 reads/day — more than enough for your house!
